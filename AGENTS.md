# AGENTS.md instructions for zmk-config-roBa

<INSTRUCTIONS>
ユーザーに対する質問は日本語で行うこと。

キーマップ、トラックボール、スクロール、マウスボタン、レイヤーなど、入力操作の体感が変わる設定を編集する前に、必ず以下をユーザーへ日本語で説明すること。

- 何を変更するのか
- その変更で何が改善すると期待できるのか
- どんなデメリットや副作用があり得るのか

ユーザーはコード自体を読まない前提で、コード名や差分だけではなく、操作感として何が変わるかを説明すること。

上記の説明後、ユーザーが明確に承認するまでは実装・設定変更・コミット・pushへ進まないこと。ユーザーが内容を理解して承認したことを確認してから作業すること。

ユーザーが作業を承認した場合は、可能な範囲で実装、検証、コミット、push、CIやビルド結果の確認まで行い、最後に何を行ったかと残っている注意点を報告すること。ユーザーがpush不要またはコミット不要と明示した場合はそれに従うこと。

GitHub ActionsやCIの実行結果を確認した場合は、最後の報告に該当するGitHub ActionsのURLを必ず含めること。リンクを取得できなかった場合は、その理由を報告すること。
</INSTRUCTIONS>
